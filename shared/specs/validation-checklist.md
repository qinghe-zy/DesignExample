# Validation Checklist

Use this checklist for major template milestones.

## Root Factory Validation

- required docs exist
- root structure is coherent
- shared conventions are discoverable
- blueprint docs exist

## Project Validation

- backend builds or clearly documents why not
- frontend builds or clearly documents why not
- SQL initialization files exist
- startup instructions are accurate
- module boundaries are documented

## Change Documentation Validation

- Plan.md updated
- Documentation.md updated
- DECISIONS.md updated for meaningful architectural choices
- CHANGELOG.md updated for notable milestones
- HANDOFF.md updated before the run closes
