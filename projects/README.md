# projects

`projects/` 用于存放当前仓库中的具体项目实例。

## 当前项目分组

### 1. 基础底座项目

- `00-base-admin`

它是当前共享基础底座，负责提供统一的管理端起点。

### 2. 三个重点项目

- `student-management`
- `library-management`
- `exam-system`

这三个项目当前已经达到较深实现状态，是后续继续推广模板工厂能力的主要参考对象。

### 3. 七个强化脚手架项目

- `mall-system`
- `blog-cms-forum`
- `hotel-booking`
- `hr-payroll-attendance`
- `course-learning-platform`
- `property-rental-dorm`
- `erp-inventory-sales`

这些项目当前不是空目录，而是具备：

- 项目 README
- 后端 / 前端 README
- 模块说明
- 启动建议
- 继承说明
- SQL/schema 基线
- 代表业务模块骨架

## 当前状态说明

`projects/` 中的项目并非全部处于同一成熟度：

- `00-base-admin` 与三个重点项目：可构建、已验证
- 其余七个项目：强化脚手架状态，便于继续派生推进

因此，阅读和使用时应区分“当前已验证”和“当前仅具备可继续推进条件”。
