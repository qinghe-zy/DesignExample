# 基础管理端蓝图

## 蓝图定位

`base-admin-template` 是当前管理端类项目的基础蓝图。

它用于为后续项目提供统一起点，而不是限制所有项目最终都长成同一种形态。

## 当前覆盖范围

- 后端基础分层
- 前端管理端基础结构
- JWT + RBAC 基线
- 仪表盘基线
- 用户 / 角色 / 菜单基线
- 通用 CRUD 页面基线
- 文件上传 / 操作日志 / 图表统计扩展点说明

## 当前适用场景

优先适用于：

- 学生管理类系统
- 图书馆类系统
- 考试管理类系统
- 其他以“后台管理”为核心的教学项目

## 当前限制

如果某类项目长期明显偏离当前管理端模式，应考虑新增兄弟蓝图，而不是一味把差异硬塞进这里。
