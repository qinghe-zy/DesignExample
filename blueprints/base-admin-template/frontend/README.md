# 前端蓝图说明

## 当前推荐结构

```text
frontend/
├─ src/
│  ├─ api/
│  ├─ components/
│  ├─ layouts/
│  ├─ router/
│  ├─ stores/
│  ├─ styles/
│  ├─ utils/
│  └─ views/
├─ index.html
├─ package.json
└─ vite.config.ts
```

## 当前能力基线

- 登录页
- 管理端布局
- 仪表盘
- 系统管理页面
- 至少一个代表业务模块页面

## 当前界面要求

- 保持管理端风格统一
- 避免无意义炫技
- 便于后续项目继续派生
